# Дорожная карта фич Near

Список идей «киллер»-уровня и статус реализации. Идём сверху вниз.

| # | Фича | Статус |
|---|------|--------|
| 1 | **Живая доска** — WebSocket на странице проекта: синхронизация задач между вкладками и клиентами без перезагрузки | **Готово** |
| 2 | **Два мира одной задачи** — канбан и ноды как две проекции одного процесса (зависимости / поток в нодах, статусы в канбане) | **Готово** |
| 3 | **Режим фокуса** — «Что делать сейчас?»: одна следующая задача по правилам (дедлайн, колонка, давность, приоритет) | **Готово (MVP)** |
| 4 | **Комментарии и таймлайн** — UI к комментариям в БД, @упоминания, лента активности по задаче | **Готово (комментарии + панель)** |
| 5 | **Сценарные пресеты** — расширение `kind`: шаблоны процессов (онбординг, релиз…), подсказки в колонках, чеклисты по умолчанию | **Готово** |
| 6 | **Офлайн-очередь** — Service Worker + очередь мутаций, синхронизация при появлении сети | **Готово (MVP)** |
| 7 | **Публичная доска «только чтение»** — ссылка без логина для стейкхолдеров | **Готово (MVP)** |

- [x] URL WebSocket с учётом `VITE_API_BASE_URL` и dev-прокси `/api`
- [x] Прокси Vite: поддержка `ws` для `/api`
- [x] Подписка на `/ws/{project_id}` на `ProjectBoardPage` (после успешной загрузки проекта)
- [x] Обработка `task_created` / `task_updated` — перезагрузка списка задач
- [x] Обработка `task_deleted` — удаление из локального состояния + событие для сайдбара

- [x] Переподключение с задержкой при обрыве соединения
