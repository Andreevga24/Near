\# 🧭 Near — Project Tracker



\## Статус: 🟢 Phase 1 / In Development



\---



\## 📌 Общее описание

Near — веб-приложение для управления проектами (аналог Yougile/Projecto/Weeek).  

Аудитория: русскоязычная. Язык интерфейса: русский + английский переключатель.



\---



\## 🧱 Технологический стек (Phase 1)

| Компонент | Технология |

|-----------|-------------|

| Frontend | React + TypeScript + Tailwind CSS |

| Backend | Node.js + Express (или Python FastAPI — уточнить) |

| DB | PostgreSQL + Prisma ORM |

| Auth | NextAuth.js / Supabase Auth |

| Деплой | Vercel (front) + Render/Supabase (back) |



\---



\## ✅ PHASE 1 — Базовый MVP (2-3 недели)



\### Модуль 1: Проекты

\- \[ ] Создание проекта (название, описание, цвет)

\- \[ ] Список проектов (карточки)

\- \[ ] Редактирование / удаление проекта

\- \[ ] Приглашение участников по email



\### Модуль 2: Задачи

\- \[ ] Создание задачи (название, исполнитель, дедлайн, приоритет)

\- \[ ] Доска задач (канбан: To Do / In Progress / Done)

\- \[ ] Перетаскивание задач между колонками (drag-n-drop)

\- \[ ] Комментарии к задаче

\- \[ ] Уведомления @username в комментариях



\### Модуль 3: Пользователи

\- \[ ] Регистрация / вход (email + пароль)

\- \[ ] Профиль пользователя (аватар, имя, роль)

\- \[ ] Роли: Админ / Участник / Наблюдатель



\### Модуль 4: Интерфейс

\- \[ ] Русский / Английский переключатель (i18n)

\- \[ ] Тёмная / светлая тема

\- \[ ] Адаптив под мобильные устройства
